# Identité de l'application

## Nom officiel

Le nom officiel de l'application est **FSAO Iris**.

## Signification

**FSAO** signifie **Fonctionnement des Services Assistée par Ordinateur**.

FSAO Iris est une application web industrielle destinée à centraliser le fonctionnement des services, la maintenance, les demandes d'intervention, les équipements, le préventif, les données QHSE, les tableaux de bord, les capteurs et les modules connectés.

## Règle de nommage

Pour conserver une présentation cohérente du projet, les intitulés visibles doivent utiliser **FSAO Iris**.

À éviter dans les nouveaux textes de présentation :

- GMAO Iris, sauf lorsqu'il s'agit explicitement du nom historique du dépôt GitHub ou d'un script déjà nommé ainsi ;
- FSAO Atlas ;
- GMAO Atlas.

## Nom du dépôt

Le dépôt GitHub peut conserver le nom technique **GMAO**, car il sert d'identifiant historique de repository :

`Kinder0083/GMAO`

Ce nom de dépôt ne remet pas en cause le nom applicatif officiel : **FSAO Iris**.
